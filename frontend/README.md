# NAS Dashboard

A modern, responsive web dashboard for monitoring and accessing services running on a Network Attached Storage (NAS) server. Built with React, Vite, and Tailwind CSS.

## Features

- **Service Monitoring**: Real-time health checks for all configured services
- **Visual Status Indicators**: Active services are highlighted with green styling, inactive ones appear dimmed
- **Quick Access**: Click any service card to directly access the service in a new tab
- **Responsive Design**: Optimized for desktop and mobile devices with large, prominent service icons
- **Server IP Display**: Dynamically fetches and displays the current public IP address using ipify API
